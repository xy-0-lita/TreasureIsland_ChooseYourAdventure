1. Clone this repository:
```bash
   git clone https://github.com/xy-0-lita/TreasureIsland_ChooseYourAdventure.git
   cd TreasureIsland_ChooseYourAdventure
```

2. Run the game:
```bash
   python play_adventure.py
```

3. Follow the prompts and make your choices carefully!

## Requirements

- Python 3.x

## Game Structure

The game uses conditional logic (if/elif/else) to create branching narratives. Players make choices at key decision points:
- **First choice**: Left or right at the fork
- **Second choice**: Wait or swim at the lake
- **Final choice**: Which colored door to enter

Only one path leads to victory!
